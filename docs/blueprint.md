# **App Name**: Ethereal Echoes

## Core Features:

- Animated Hero Section: A visually appealing hero section with a gentle animated gradient background and her name.
- Personal Letter Section: Display a heartfelt handwritten-style letter to convey personal feelings and memories.
- Interactive Life Timeline: Showcase significant life moments in a smooth, vertically scrolling timeline.
- Expandable Notes Section: Display clickable note cards that smoothly expand into modals to reveal more details.
- Cherished Memories Carousel: Showcase precious memories through an image carousel with smooth transitions and captions.
- Guestbook Messages: A form connected to Firestore database where people can submit their birthday wishes and messages.
- Closing Thoughts Section: Concluding section with emotional promises, an LLM and personalized farewell message.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke calmness and elegance.
- Background color: Off-white (#F8F8FF) for a clean and airy feel.
- Accent color: Peach (#FFDAB9) for warm highlights and emotional connection.
- Headline font: 'Alegreya', a serif with an elegant, intellectual, contemporary feel.
- Body font: 'PT Sans', a clean, humanist sans-serif, for readability and warmth.
- Minimal and gentle icons that complement the soft aesthetic, in lavender and peach hues.
- Use of rounded cards, soft shadows, and a minimalist layout for an intimate and calming user experience.
- Smooth, Framer-motion-style fade and slide animations to enhance user engagement and create an intimate, emotional feel during vertical scrolling.
